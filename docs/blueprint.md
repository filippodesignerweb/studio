# **App Name**: LED 4U

## Core Features:

- Website Content Display: Render detailed information about LED 4U's services, projects, company profile, and FAQ in a well-structured, responsive layout.
- Service Inquiry Form: Provide a form for users to request technical evaluations or custom project quotations.
- AI-Powered Project Assistant Tool: Based on user inputs in the inquiry form, a generative AI tool offers initial recommendations for LED panel solutions or estimated project scope, aiding the sales process.
- Interactive FAQ: An expandable FAQ section addressing common client questions, improving user self-service.
- Project Portfolio Showcase: Display a visual gallery of completed projects with relevant images and details to highlight expertise.

## Style Guidelines:

- Color Palette: Dark background '#0E0D12' (very dark purple-gray), with primary accent of vibrant Teal '#029E9D' and secondary accent of bold Purple '#9800FF'. These colors are used in dynamic gradient texts, emphasizing a high-tech and modern brand identity.
- Headline font: 'Unbounded' (sans-serif) for high-impact titles, creating a bold and distinctive presence. Body font: 'Bricolage Grotesque' (sans-serif) for readable and modern text. Note: currently only Google Fonts are supported.
- Minimalist and clean line icons for navigation, mobile menu, social media, and FAQ accordions, consistent with a tech-oriented, modern aesthetic.
- Responsive layout designed for optimal viewing across various devices. Content is centered within wide sections with clear padding, using CSS Grid and Flexbox for organized arrangement. Employs parallax scrolling and distinct section transitions for a dynamic user experience.
- Extensive use of GSAP and ScrollTrigger for sophisticated animations, including hero parallax effects, section transitions, a smooth logo marquee, timeline progress indicators, and dynamic text gradients for a futuristic feel. Subtle hover effects on interactive elements enhance engagement.
